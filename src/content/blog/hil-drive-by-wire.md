# Hardware-in-the-Loop (HIL) Drive-by-Wire

This project demonstrates a Hardware-in-the-Loop (HIL) testbench designed for a drive-by-wire automotive steering system.

## Overview
Drive-by-wire replaces traditional mechanical linkages with electronic sensors and actuators. Testing these systems safely requires extensive HIL simulation where the electronic control unit (ECU) interacts with a mathematical model of the vehicle.

## Technologies Used
- MATLAB/Simulink for vehicle dynamics modeling
- Real-time computing hardware
- CAN bus communication protocols

## Outcomes
Successfully validated the ECU's fault-detection algorithms under simulated sensor failures. The HIL setup allowed for comprehensive test coverage without the risks and costs associated with real-vehicle testing.
